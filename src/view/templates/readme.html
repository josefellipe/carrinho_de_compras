Minha Loja Backend

Seja Bem-vindo a esta aplicação! Sendo um desenvolvedor focado no Backend, a aplicação foi criada para mostrar funcionalidades do backend, não do front. Contudo, foram adicionadas algumas páginas para que o usuário pudesse testar tais funcionalidades.:

Funcionalidades Principais:

  • Sistema de Carrinho de Compras: Os usuários podem adicionar produtos ao carrinho e realizar compras.
  • Cadastro de Pessoas: Os usuários podem se cadastrar na plataforma e realizar compras.
  • Cadastro de Produtos: Administradores podem adicionar e gerenciar os produtos disponíveis na loja.
  • Diferenciação de Poder entre Admin e Usuários: Administradores têm acesso a recursos adicionais e permissões que usuários normais não têm.

Recursos Futuros:

  • Controle de Estoque: Implementar um sistema de controle de estoque para gerenciar a disponibilidade dos produtos.

Ferramentas:

  • FastAPI: A aplicação foi criada usando o fastAPI em Python e roda em uvicorn.
  • SQLite: Para simplificar, o banco de dados foi criado usando o SQLite.
  • SQLAlchemy: Como ORM estou usando o SQLAlchemy, tanto para facilitar a troca de banco de dados futuramente, quanto para aumentar a camada de segurança.
  • Criptografia: Será adicionada uma camada de criptografia para que os dados sejam armazenados criptografados.

Espero que você curta o projeto e que ele possa ser útil de alguma forma!
